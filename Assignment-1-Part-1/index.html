SWE-645 Assignment 1

SWE 645 (Component-based Software Development) covers some of the topics related to the software development models that are used to support component-based software systems.The course will largely focus on server-side software design and development. Teaches technologies such as Microservices, Docker Containers, Container Orchestrator - Kubernetes, DevOps – CI/CD, Infrastructure as Code – Terraform, Configuration Management Using Ansible, GitOps, Resilient Architectures in AWS cloud, as well as a range of AWS services ranging from compute, networking, storage, and security. Students will have options to do homework assignments/project on Amazon Web Services (AWS) and/or Google Cloud Platform (GCP).
